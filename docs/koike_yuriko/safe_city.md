# 公約: セーフシティ
---------------------

以下、[東京大改革3.0ホームページ](https://www.yuriko.or.jp/policy)より引用

- 地震・火災から命を守る木造住宅密集地域の解消促進
- 新たな「調節池」の整備で水害に強いまちづくり
- 災害時、道路を活かし命を守る「無電柱化」の推進
- 900万人が暮らす「マンション防災」の強化
- 「シェルター整備」でミサイルの危機から都民を守る
- 安全・安心で快適な「避難所改革」の推進
- 富士山噴火を想定した降灰対策の実施
- グリーンインフラでまちの保水力を上げる
